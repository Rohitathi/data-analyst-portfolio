## Project 1: End-to-End Streaming Data Pipeline
**Stack:** Kafka, AWS S3, PySpark, PostgreSQL

A fully automated pipeline that ingests real-time sales events from Kafka, stores raw data in AWS S3, transforms with PySpark, and loads into PostgreSQL for reporting.

- Processed 100 real-time events end-to-end
- Regional sales aggregations with avg deal size
- Automated business report from PostgreSQL
